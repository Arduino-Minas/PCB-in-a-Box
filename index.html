Pcb-in-a-box

Confeccionar placas é uma das principais práticas de quem desenvolve protótipos em eletrônica, principalmente quem trabalha com Arduino, seja algo simples ou mais elaborado. Então criamos este projeto para permitir que todo profissional ou hobista, estudante ou amador possa desenvolver seu PCB facilmente usando o processo por transferencia por Luz ultra violeta.

View project onGitHub

Bem Vindo ao PCB in a BOX.

Este projeto visa desenvolver um base de insolação por LEDs Ultravioleta para confecção de PCBs por métodos similares ao de SilckScreen.
